# patch-building-with-OTSU-for-public-dataset
This project is an implementation of the OTSU algorithm for masked segmentation of slide-level pathology image public datasets and generation of patches.

    numpy                1.21.5

    matplotlib           3.5.1
    pytorch              1.12.0
    tqdm                 4.64.0
